#<a href="http://www.hecatonfish.com/justsynth">Just Synthesizer</a>

###To Install

Download the executable and run it. This program requires Java to be installed.

###To Use:

|Key:					|	Effect
|---------------------------------------|----------------------------------------------------------
|1-BACKSPACE and TAB–\			|	Play Notes
|A–ENTER				|	Retune the Keyboard to a Note
|Z					|	Select Equal Tempered Tuning
|X					|	Select Just Tuning (7 Limit)
|C					|	Select Just Tuning (5 Limit)
|V					|	Select Just Tuning (Symmetric)
|B					|	Select Pythagorean Tuning
|LSHIFT					|	Display Table in Cents or as Ratios
